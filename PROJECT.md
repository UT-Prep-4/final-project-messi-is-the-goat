#Python Plants VS Zombies

## What it is
This program is a python port of plants and zombies. It uses the graphics libary
pygame to do this. We used classes to structure the code into multiple sprites 
which are resuable and enable the code to maintain a clean format. 

## How to run it
Run python final_project.py in the terminal, then open port 6080 to see the game
window. Use keys 1-4 to select the plant you want to place (1->Sunflower, 2->
Peashooter, 3->Wallnut, 4->Potatomine). Then click the Suns to get the suns.
Each plant costs 20 suns.

## How it works
We made a class for each of the plants, each class has functions, like update, 
end, and clicked. These allow for the code to follow a clean pattern and
structure. As well there is a class for the zombies that can be called multiple
times to make multiple zombies.
